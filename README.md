## ✨ Features

* User Authentication
* JWT Authorization
* Role-Based Access
* Sweet Inventory Management
* Purchase Management
* Sales Tracking
* PostgreSQL Database
* RESTful APIs
* Responsive Design

## ⚙️ Installation

### Clone Repository

git clone YOUR_REPOSITORY_URL

### Backend Setup

cd backend

npm install

npm run dev

### Frontend Setup

cd frontend

npm install

npm run dev
