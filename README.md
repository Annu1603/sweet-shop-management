## ✨ Features

- User Authentication
- JWT Authorization
- Role Based Access Control
- Sweet Inventory Management
- Purchase Tracking
- Sales Tracking
- REST APIs
- Responsive UI

## 🛠️ Tech Stack

### Frontend
- React.js
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express.js
- PostgreSQL

### Deployment
- Vercel
- Railway

## 📂 Project Structure

backend/

## ⚙️ Installation

### Clone Repository

git clone YOUR_REPO_LINK

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev
